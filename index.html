Reflexión

¿Qué aprendí? He aprendido que los componentes son la esencia de la construcción de interfaces de usuario en React. Actúan como bloques de construcción independientes y reutilizables, cada uno representando una unidad lógica bien definida de la interfaz. Pueden ser de dos tipos: componentes de clase, que manejan estados y son más complejos, o componentes de función, más simples y centrados en la representación visual. Cada componente se subdivide en código HTML para la estructura, JavaScript para el comportamiento, y CSS para el estilo. ¿Cómo aprendí? Aprendí mediante el análisis de las definiciones y procedimientos descritos en las fuentes proporcionadas. La información sobre la estructura de los componentes, los tipos de componentes y las características de JSX me permitieron comprender cómo funcionan y cómo se relacionan entre sí. La descripción de las expresiones, condicionales y bucles en JSX me ayudó a entender cómo se puede usar para crear componentes dinámicos y flexibles.

¿Qué aprendí? • La capacidad de crear componentes reutilizables es fundamental en React. Esto permite dividir la interfaz de usuario en piezas independientes y reutilizables, lo que mejora la organización del código y la eficiencia del desarrollo. • Existen herramientas que simplifican la configuración del entorno de desarrollo de React. Algunas de las más populares son Create React App, Vite y Next.js. Estas herramientas facilitan la creación de un proyecto, la configuración de Webpack, Babel y otras dependencias necesarias. • La importancia de herramientas como Babel, que transpila código JavaScript moderno a versiones compatibles con navegadores antiguos, ESLint, que analiza el código en busca de errores y problemas de estilo, y Webpack, que empaqueta y optimiza los archivos de la aplicación para su despliegue. ¿Cómo aprendí?He aprendido sobre React a través del análisis del documento de la clase, este documento ofrece una visión general de React, su historia, sus características principales, las herramientas utilizadas para el desarrollo con React y las diferentes formas de crear un proyecto. Reflexión Metacognitiva

¿Qué aprendí durante este curso?

Durante este curso de Desarrollo de Aplicaciones Web, he experimentado un crecimiento significativo en mis conocimientos sobre tecnologías web fundamentales. Desde la primera semana, donde aprendí los conceptos básicos de las tecnologías web y los diferentes roles en el desarrollo, hasta las últimas semanas donde dominé frameworks como Bootstrap y Tailwind CSS.

Más allá de las tecnologías específicas como HTML5, CSS3, y los frameworks, desarrollé una comprensión profunda sobre la estructura del DOM, la importancia del diseño responsivo, y cómo crear interfaces de usuario modernas y accesibles. También aprendí sobre las mejores prácticas en el desarrollo web y la importancia de escribir código limpio y mantenible.

¿Cómo aplicaré estos conocimientos?

Los conocimientos adquiridos en este curso serán fundamentales para mi desarrollo profesional como Ingeniera de Sistemas. Planeo aplicar estas habilidades en proyectos reales, tanto académicos como profesionales, creando aplicaciones web que resuelvan problemas específicos en mi área de interés: tecnologías de la información y ciberseguridad.

También utilizaré estas habilidades para crear herramientas web que apoyen mis otros intereses como el hacking ético y la continuidad de TI. Además, estos conocimientos me servirán como base para continuar aprendiendo tecnologías más avanzadas como JavaScript, React y desarrollo backend.
